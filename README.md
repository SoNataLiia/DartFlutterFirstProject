# Tarea 2 - Flutter

## Descripción
Aplicación Flutter con dos campos de texto y un botón.

## Funcionalidad
- El usuario introduce dos números en los campos de texto.
- Al pulsar el botón "Calcular suma", se muestra el resultado de la suma de los dos números.
- Si algún campo está vacío o el valor no es numérico, se muestra un mensaje de error.

## Tecnologías
- Flutter
- Dart
- StatefulWidget
- TextEditingController

## Requisitos
- Flutter SDK 3.0 o superior
- Dart SDK
